# Escape Room Game

## Structure
- All images are in `assets/images/`
- Audio files are in `assets/audio/` and `assets/effect/`
- Scripts are in `scripts/`
- HTML levels are in `levels/`

## How to Run
Open `index.html` in your browser to start the game.

---

This project is a browser-based 2D escape room game. Use arrow keys to move, avoid skeletons, collect the key/hammer, and solve riddles to win!
